# SQL-Assignment-6---Employee-Project-Management-Database-Analysis
## 📌 Overview
This assignment focuses on employee and project management using EmployeeDetail and ProjectDetail tables. 
It covers SQL joins, aggregate functions, grouping, and project analysis.

## 🚀 Topics Covered
- Employee and Project Tables
- Aggregate Functions (SUM, AVG, MIN, MAX)
- GROUP BY & HAVING
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- CROSS JOIN
- Project-wise Analysis
- Department-wise Salary Analysis

## 🎯 Learning Outcomes
- Understanding table relationships
- Working with joins and grouping
- Performing department and project analysis
- Writing complex SQL queries
